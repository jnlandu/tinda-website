## ğŸš€ Getting Started

### Prerequisites
- **Node.js** 18.0 or higher
- **npm** or **pnpm** (recommended)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/tinda/tinda-website.git
   cd tinda-website
   ```

2. **Install dependencies**:
   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

4. **Open browser**:
   Navigate to `http://localhost:3000`
